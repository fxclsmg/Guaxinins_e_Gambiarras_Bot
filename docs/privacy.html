Política de Privacidade

Início Termos de Serviço

Última atualização: 11 de julho de 2026

Dados utilizados

  • ID do usuário.
  • ID do servidor.
  • ID do canal.
  • Atributos do sistema Guaxinins & Gambiarras.
  • Estado dos baralhos.
  • Dados temporários necessários ao funcionamento.

Finalidade

Os dados são utilizados exclusivamente para o funcionamento do bot.

Compartilhamento

Nenhum dado é vendido ou compartilhado com terceiros.

Retenção

Os dados podem ser removidos pelos comandos do próprio bot ou automaticamente quando deixarem de ser necessários.

Segurança

São adotadas medidas razoáveis para proteger os dados armazenados.
