Sobre

O Guaxinins & Gambiarras Bot foi desenvolvido para suprir minha necessidade de um bot no Discord que saque cartas de um baralho de 52 cartas. Além disso posso estudar e brincar com o que amo, o RPG. Aliás ouçam Realidades Paralelas do Guaxinim.

Entre suas funcionalidades estão:

  • Rolagem de dados tradicionais.
  • Rolagem para o sistema Guaxinins & Gambiarras.
  • Rolagem das doze notas musicais.
  • Gerenciamento de baralhos de cartas.
  • Comandos auxiliares para jogos.
